# Scoop Bucket for OpenPencil

Official Scoop bucket for installing OpenPencil on Windows.

This repository contains the Scoop manifest used to distribute OpenPencil, the open-source AI-native vector design tool from ZSeven-W.

## Install

```powershell
scoop bucket add openpencil https://github.com/ZSeven-W/scoop-openpencil
scoop install openpencil
```

## Upgrade

```powershell
scoop update openpencil
```

## Uninstall

```powershell
scoop uninstall openpencil
```

## About OpenPencil

OpenPencil is a design-as-code canvas for turning prompts into live UI. Learn more here:

- Main repo: https://github.com/ZSeven-W/openpencil
- Website: https://op.zseven.tech
